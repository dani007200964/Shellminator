This software runs even on the weakest hardware!

When creating Shellminator, our main goal was to support as many microcontrollers as possible. It works out of the box on every Arduino board we tested, and for the hardcore developers out there, it's easy to port to any environment. The only requirement?
The project must be written in C++.

How much functionality you can use at once depends only on the available memory. You can do amazing things even with an old Atmega328-based Arduino Uno R3, but if you want to unlock the full potential of Shellminator, we recommend using an Arduino Uno R4, a Raspberry Pi Pico or an ESP32—these platforms come with a lot more memory.

In the image above, you can see an Arduino Nano (with the older Atmega328 processor) connected to an Android phone via the Type-C port. The Arduino Nano handles the task effortlessly, and with any serial port app on the smartphone (in this case, USB Terminal ), you can connect to Shellminator in no time.

This can be super handy in the field, but it also makes development and maintenance much easier!

We've already integrated it into robots, IoT sensors, audio amplifiers, and even a web browser so you can try it out for yourself. Don’t believe it? See for yourself here.
