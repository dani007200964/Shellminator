Arduino: One of the Most Popular Development Platforms Today

Arduino is one of the most widely used development platforms, found everywhere from the film industry to university research labs. One of the main reasons for its success is its simplicity—much easier to use than Eclipse-based systems, for example. You grab an Arduino-compatible board, open the software, select your board, maybe install a few cool libraries, and boom—you’re ready to start coding!

We love this fresh and dynamic approach too, which is why Shellminator is officially available in the Arduino Library Manager. That means you can install it in just a few clicks using the Arduino IDE.

Of course, not everyone loves the convenience of the Arduino IDE. If you’re more of a hardcore PlatformIO fan, no worries—Shellminator is fully supported there too and just as easy to install.

And for the next-level hardcore developers who prefer Eclipse-based environments like STM32 CubeIDE, don’t worry—you can still use Shellminator there as well. It won’t be a two-click install, but if you’re a seasoned terminal ninja, you’ll get it up and running in no time.
