Detailed Description

ShellminatorBufferedPrinter object.

It can be used to accelerate the rendering process on the host terminal interface. Terminal emulators usually works faster, when the data arrives in one large piece. Otherwise some annoying effect can be observed like flickering or cursor jumping. This object aims to fix this problem. It is a simple buffered printer.

Allocate memory for the internal buffer.

With this function, you can allocate memory for the internal buffer.

Parameters
bufferSizeAllocated memory in characters. It must be at least 30 characters!
Returns
If the memory allocation is done it will return true. If it fails for some reason, it will return false.
Warning
To use the object, this function must be called in the init section.
